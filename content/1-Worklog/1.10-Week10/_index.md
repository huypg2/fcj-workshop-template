### Week 10 Achievements:

* Understood how to build a document management system using Serverless architecture with Lambda, DynamoDB, API Gateway, Amplify, and CloudFront.
* Understood how CRUD functions can be built with Lambda and DynamoDB so the system can create, read, update, and delete document information.
* Understood the role of AWS Amplify in supporting frontend development, storage, authentication, and faster connection to Serverless backends.
* Understood how API Gateway receives requests from the frontend, routes them to Lambda, and returns results to users.
* Learned how AWS SAM deploys Serverless resources through templates, making system updates clearer and easier to control.
* Understood the role of CloudFront in faster content delivery, latency reduction, and better document access experience.
* Understood how search features help users look up documents by name, content, or related information.
* Understood how X-Ray, CloudWatch, and Grafana support request tracing, logs, metrics, system error analysis, and operational dashboards.
* Summarized approaches for building Serverless Web Apps, Serverless APIs, and Serverless chat applications, showing how AWS services work together to create complete applications.
* Understood the role of Elastic Beanstalk in fast application deployment, where AWS supports runtime environment, load balancing, scaling, and monitoring.
* Learned the workflow for deploying Node.js applications on AWS, including source code preparation, environment configuration, and post-deployment testing.
* Understood how CI/CD with Elastic Beanstalk and CDK Pipelines automates application updates, reduces manual work, and limits release errors.
* Understood WordPress architecture on AWS, including EC2 servers, database, storage, and required configurations for website operation.
* Understood principles for building highly available web applications such as load balancing, auto scaling, multi-AZ, and system monitoring.
* Learned more about Windows Server HA and SQL Server HA on AWS, understanding how enterprise systems reduce downtime and improve recovery during incidents.
